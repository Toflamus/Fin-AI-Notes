# A 股做市商制度现状

在中国的 A 股市场，做市商并非像美股纳斯达克那样覆盖全市场，而是以**"竞价交易为主、做市交易为辅"**的混合模式存在。

做市商在特定板块（科创板、北交所）和衍生品市场（ETF、期权）中占比极高，但在主板（沪深两市普通股票）的参与度相对有限。

---

## 一、做市商的占比与分布

截至 2026 年，A 股做市制度主要集中在以下四个领域：

| 细分市场 | 做市商制度现状 | 占比/活跃度情况 |
|---------|--------------|----------------|
| **科创板** | 混合做市制度 | 超过 20 家券商获得做市资格，覆盖面已从初期几十只扩展到数百只 |
| **北交所** | 混合做市制度 | 北交所扩容明显，总市值逼近万亿元。做市商成交占比约 **15%-20%**，是提供流动性的绝对核心 |
| **ETF & 期权** | 全面做市制度 | 最活跃的领域。ETF 市场 **90% 以上**的交易流动性由做市商维护；期权市场（如上证 50ETF 期权）做市商参与度几乎 **100%** |
| **主板（沪深）** | 竞价交易为主 | 传统主板股票**基本不实行做市制度**，依然靠股民和机构的限价单进行撮合 |

---

## 二、谁是主要的做市商？

目前 A 股的做市商**几乎清一色由头部证券公司（券商）及其子公司担任**。

### 第一梯队（AA 级 / 头部玩家）

这些券商拥有最强大的算法交易系统、资金实力和风险控制能力：

| 券商 | 特点 |
|------|------|
| **中信证券** | 龙头，覆盖范围最广 |
| **中信建投** | 综合实力强劲 |
| **华泰证券** | 技术实力强，量化做市领先 |
| **国泰君安** | 全牌照综合券商 |
| **招商证券** | 稳健的全国性券商 |
| **东方证券** | 在期权做市领域表现突出 |
| **海通证券** | 老牌头部券商 |

### 第二梯队（活跃玩家）

| 券商 | 特点 |
|------|------|
| 广发证券 | 华南龙头 |
| 银河证券 | 国有大型券商 |
| 国信证券 | 深耕深圳市场 |
| 申万宏源 | 研究实力突出 |
| 中金公司 | 主要服务于大盘蓝筹和跨境 ETF |
| 浙商证券、财通证券 | 深耕特定区域或板块 |

---

## 三、A 股做市商的特殊性

### 1. "不改变竞价本质"

与美股做市商不同，A 股做市商是在既有的订单簿 (Order Book) 里**插队提供买卖报价**。他们起到的是"润滑剂"作用，**而不是唯一的交易对手方**。

散户和机构的订单仍然直接进入中央订单簿参与撮合，做市商只是在盘口薄弱时补充双边报价。

### 2. 盈利模式

主要通过：
- **买卖价差 (Spread)**：在买一和卖一之间赚取微小差价
- **交易所的返佣 / 手续费减免**：交易所为鼓励做市而提供的激励

### 3. 技术门槛

目前的做市竞争本质上是**量化高频交易 (HFT) 的竞争**。头部券商（如华泰、中信）投入了巨额资金在：
- 极低延迟的交易系统
- FPGA 硬件加速
- 微观结构预测模型
- Co-location（机房托管）

## 四、对不同投资者的影响

| 关注板块 | 做市商的角色 |
|---------|-------------|
| **科创板/北交所小盘股** | 做市商占比正在快速提升，是防止"闪崩"和提高成交效率的关键。流动性高度依赖做市商 |
| **主板大盘股**（上证 50 / 沪深 300） | 依然是散户和公私募机构之间的订单博弈，做市商身影主要出现在配套的 **ETF 和期权合约**中 |
| **ETF / 期权** | 做市商是绝对核心，没有做市商这些产品几乎无法交易 |
