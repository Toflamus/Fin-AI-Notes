## 一、A 股集合竞价完整规则

### 1.1 早盘集合竞价（9:15-9:25）

| 时间 | 规则 |
|------|------|
| **9:15-9:20** | 可挂单、**可撤单** |
| **9:20-9:25** | 可挂单、**不可撤单** |
| **9:25** | 统一撮合，产生**开盘价** |

**成交价原则**：由**最大成交量原则**决定。所有成交均以**同一价格**成交。

### 1.2 收盘集合竞价（14:57-15:00）

| 时间 | 规则 |
|------|------|
| **14:57-15:00** | 可挂单、**不可撤单** |
| **15:00** | 统一撮合，产生**收盘价** |

**关键特征**：
- 进入 14:57:00 后，**连续竞价阶段遗留的未成交单也无法撤销**
- 统一以**收盘价**成交，**不会比挂单价更差**（买单 ≤ 收盘价，卖单 ≥ 收盘价）
- 收盘价由集合竞价撮合产生，**而非最后一笔成交价**

### 1.3 ⚠ 未成交单的顺延规则（关键机制）

| 阶段 | 未成交单子去向 |
|------|--------------|
| 早盘集合竞价（9:25 前） | **自动顺延**进入连续竞价（9:30） |
| 连续竞价（9:30-14:57） | **自动顺延**进入收盘集合竞价 |
| 收盘集合竞价（14:57-15:00） | 未成交则当日**自动作废** |

> **深交所**明文规定连续竞价单顺延至收盘集合竞价；**上交所**表述为"进入后续竞价阶段"，实际效果相同。

**这条规则是整篇文章的核心伏笔**：

**机构如果什么都不做，上午 9:30 挂的限价单如果一直没成交，会自动被"拖入"14:57 的收盘集合竞价——被 3 分钟黑箱吞噬**。

所以机构必须**主动撤单**，而不能"懒得管"。这是 14:56:59 撤单风暴的制度根源。

### 1.4 全时段速查表

| 时间段 | 状态 | 可否撤单 |
|-------|------|---------|
| 09:15-09:20 | 开盘集合竞价（可撤单阶段） | ✅ |
| 09:20-09:25 | 开盘集合竞价（不可撤单） | ❌ |
| 09:25-09:30 | 休市 | — |
| 09:30-11:30 / 13:00-14:57 | 连续竞价 | ✅ |
| **14:57-15:00** | **收盘集合竞价（不可撤单）** | **❌** |

### 1.5 注意事项

- 收盘集合竞价以**统一价格**成交，不会比挂单价更差，但**不代表"一定赚钱"**——盈亏取决于持仓成本和未来走势
- 想避免单子被带入收盘集合竞价，必须在 **14:57 前主动撤单**
- **日内交易者、不想持仓过夜者需特别注意这个时间节点**

**核心约束**：进入 **14:57:00** 后，任何在单簿中的挂单**无法撤回**，只能等到 15:00 一起按统一价撮合。

**唯一的逃生窗口**：**14:56:59 这一秒是机构最后一次能撤单的时机**——过了就被绑死。

## 二、散户直觉：为什么觉得"集合竞价是白捡"？

从普通散户角度看，逻辑似乎完全合理：

> "我挂了 10.00 元的买单进入集合竞价。
> - 如果不成交 → 我不亏
> - 如果撮合出来的收盘价是 9.90 元 → 我不仅成交了，还'白赚'了 1 毛钱的优惠
>
> 这有什么不好的呢？"

这种思维的漏洞在于——**它默认"不成交"是没有代价的**，并且**默认"优惠成交"一定是好事**。

在量化微观博弈中，这两条假设**全部错误**。

## 三、机构视角：被卷入收盘集合竞价 = 三重噩梦

### 原因 1：致命的逆向选择风险 (Adverse Selection)

你以为的"**优惠成交**"，在微观结构中通常意味着"**接飞刀**"。

#### 场景推演

- 你挂了 **10.00 元**的买单，14:56 仍然挂在单簿
- 进入 14:57，你的单子被"绑死"
- 此时突然有一个游资大佬决定在尾盘**清仓**，直接砸出 **5 万手市价卖单**
- 收盘价被硬生生砸到 **9.80 元**
- 你的 10.00 元买单以 **9.80 元"优惠"**成交了

#### 量化视角的灾难

你确实买得便宜了，但这说明**整个市场的净头寸已经发生了剧烈的向下反转**。你买入的瞬间，股票正处于**巨大的抛压之中**。

**关键对比**：
- **连续竞价**：算法在看到第一笔 1000 手砸盘时**就撤单跑了**（实时 L2 反应）
- **集合竞价**：你被绑死，**只能硬接这 5 万手抛压的余波**

> **一句话总结**：**你以为你占了便宜，其实你成了别人出货的流动性垫背。**

这就是金融学里所谓的**"Winner's Curse"**——你能"赢得"这笔优惠成交的唯一原因，是因为更聪明、信息更充分的玩家正在逃离。

---

### 原因 2：无法承受的敞口/隔夜风险 (Inventory & Overnight Risk)

散户想的是"不成交罢了，今天没买到明天再买"。
对于机构，**这是极其严重的风控事故**。

#### 日内策略的死穴

很多高频/日内策略要求**每天收盘时仓位必须严格归零 (Flat Position)**：
- T+0 融券做空策略
- 统计套利（Pairs Trading）
- ETF 套利
- 期现套利（需要同步平掉期货与现货）

这些策略的数学假设是**"日内对冲完毕"**，一旦带仓过夜，整个风险模型就**不再成立**。

#### 被拒之门外的后果

如果算法挂了平仓单，但在集合竞价中因为价格变动"**没成交**"：
- 机器带着巨大的**风险敞口过夜**
- 原本 Delta-neutral 的组合变成**单边暴露**
- 风控系统会触发**紧急告警**

#### A 股 T+1 的惩罚

A 股 T+1 规则让这个风险**雪上加霜**：
- 今晚美股暴跌 / 突发黑天鹅
- 这笔没能平掉的仓位**明天一开盘就面临惨烈的跳空低开**
- T+1 规则下，**今天买入的部分当天还不能卖**——连紧急止损都做不到

#### 机构的算盘

> "我宁愿在 14:56 的时候**主动吃卖一**（承担一点滑点成本），去买一个 **100% 确定能平仓**的结果，也**绝对不敢**把命运交给充满不确定性的集合竞价。"

---

### 原因 3："三分钟失明"带来的控制权丧失

高频交易的核心优势是什么？**速度和信息更新频率**。

#### 连续竞价 vs 集合竞价的信息对比

| 维度 | 连续竞价 | 集合竞价（14:57-15:00） |
|------|---------|----------------------|
| L2 盘口更新 | **毫秒级** | 深交所 **3 秒一次**，上交所甚至**完全不播报实时撮合量** |
| 可撤单 | ✅ 任何时候 | ❌ 全程锁死 |
| 订单簿可见性 | 完整 | 只有每 3 秒的参考价，成交量盲盒 |
| 对手方识别 | 可追踪 MBO | 完全黑箱 |
| 算法调整空间 | 每毫秒自适应 | **零** |

#### 比喻

这相当于把一个**习惯开着法拉利看实时导航的赛车手**，突然**蒙上眼睛让他盲开 3 分钟**。

对于依赖实时数据的高频算法：
- 失去对订单的控制权 = **系统化交易的绝对禁忌**
- 风控系统不允许在盲区中持仓
- 所有依赖实时 L2 的策略（做市、OFI 预测、统计套利）**全部失效**

## 四、"确定性溢价"：机构的真实决策

### 金融学术语

你观察到的"**尾部抬升（14:56:59 疯狂撤单）**"，在金融学里叫做**为确定性支付的流动性溢价 (Liquidity Premium for Certainty)**。

### 机构的典型操作序列

```
14:55:00  算法开始清理被动挂单
14:56:00  持续撤单，监控单簿剩余量
14:56:50  ★ 疯狂撤单高峰 ★
14:56:58  转换：把所有仍需平仓的头寸转为市价单/扫盘单
14:56:59  最后一次撤单 / 最后一次主动扫盘
14:57:00  集合竞价开始，机构已"全员逃离"
          单簿里剩下的主要是：
          - 散户的限价单
          - 少数确实想参与收盘竞价的大宗交易
          - 被动指数跟踪盘（比如 ETF 收盘价复制）
15:00:00  集合竞价撮合完成
```

### 机构心里的算盘

> "我宁愿现在花 **2 分钱的滑点代价**，主动去买 / 卖，把仓位锁死；
> 也绝对不去 **14:57 的池子里赌那个可能"优惠"、但更可能让我万劫不复的盲盒**。"

**本质上**：机构用**确定的小滑点**去避免**概率分布为尾部极厚的大损失**——这在期权定价里就是标准的**风险溢价 (Risk Premium)**。

### 为什么 Log-Log 图上尾部尖峰如此庞大？

**在"不确定性"面前，所有的聪明钱都会选择在同一秒钟夺门而出**。

这是一个典型的**同步行为**：
- 所有专业机构面对同样的规则约束
- 所有专业机构得出同样的理性决策
- 所有专业机构的撤单时间点**高度同步**在 14:56:59

→ 在日内撤单率的时间分布图上，形成一个**尖锐的尾部峰值**。

## 五、在订单归属识别研究中的含义

这个现象是 `Fin-Research/OrderAttribution_AShare_2026Q2/` 主题中的**关键判别信号**。

### 作为机构识别的强指纹

| 时间窗口 | 观察到的行为 | 机构可能性 |
|---------|------------|----------|
| 14:56:50 - 14:56:59 撤单率突然飙升 | 日内算法/做市商大规模撤单 | **极高** |
| 14:57:00 - 15:00:00 持续挂单（未撤） | 散户 or 少数指数跟踪盘 | **极低**（机构已撤） |
| 14:56:59 大单转市价扫盘 | 机构紧急平仓 | **确定** |
| 14:58:00 仍在挂被动限价单 | 几乎肯定是散户 | **极低** |

### 可构造的因子

```
InstitutionalCancelSpike_t =
    (t ∈ [14:56:50, 14:57:00]) · 撤单数量 / 1分钟平均撤单数量
```

高值 → 机构活动活跃的股票；低值 → 机构参与度低的票。

### 与五熵因子的联系

这个尾部"撤单风暴"会让：
- **F1 SVD 熵**：在 14:56:50-14:57:00 窗口内**急剧上升**（盘口从"被机构控制的低维" → 散户主导的高维）
- **F3 Markov 熵率**：价格路径在撤单风暴后**可预测性增强**（只剩散户噪音）
- **VPIN**：可能异常——机构转市价单扫盘会制造短暂 VPIN 飙升

这是一个**可直接实证的假设**，值得在主题研究中验证。

## 七、三个常见追问的深化回答

散户直觉往往在以下三个点上与机构逻辑产生分歧。每一个都值得单独拆解。

### Q1：什么是"逆向选择风险"？

见独立笔记 [逆向选择风险](../Basics/AdverseSelection.md)——包含 Glosten-Milgrom 模型的完整推导。

**本场景的浓缩版**：

> 你挂 10.00 元买入 = 你基于"它值 10.00"这个判断在下注。
> 愿意立刻把货卖给你的人 = **他知道这股票马上要跌到 9.90**。
>
> 在收盘集合竞价里，这种不对称尤为极端：
> - 你**无法撤销**
> - 对方可能掌握**大宗数据/内幕信息**预判收盘后的利空
> - **你的挂单成为了对方的离场窗口**
>
> 你得到的不是"便宜"，而是**"信息落后带来的亏损"**。

---

### Q2："撤单了仍然平不了仓"怎么办？——撤单 ≠ 不成交

这是最容易产生的误解。机构撤单的目的**不是**"不卖了"，而是**"改变卖的方式"**。

#### 撤单后的标准操作序列

```
14:56:58.xxx  撤掉所有被动限价单（Passive Limit Orders）
14:56:58.xxx+ε  （同一毫秒内）算法根据当时盘面
14:56:59.000  发出市价单 / 激进限价单（Aggressive Order）
              直接扫掉对手方的单子
14:56:59.xxx  头寸锁死，100% 成交完毕
14:57:00      进入集合竞价黑箱——机构已不在其中
```

#### 对比两种路径

| 路径 | 成本 | 成交概率 |
|------|------|---------|
| **不撤单，任由其进入集合竞价** | 0 滑点（被动挂单价格不变） | **不确定**（可能撮合不上、可能大幅偏离） |
| **撤单重发为市价单** | **确定的滑点**（跨过价差，吃掉对手） | **100% 成交** |

#### 为什么机构选择路径 2？

核心是一个**贝叶斯决策问题**：

$$\text{Expected Loss}_{\text{path 1}} = P(\text{不成交}) \cdot \text{隔夜风险损失} + P(\text{成交}) \cdot E[\text{逆向选择损失}]$$

$$\text{Cost}_{\text{path 2}} = \text{确定的滑点}$$

**机构的算盘**：隔夜风险损失的**尾部极重**（美股暴跌、黑天鹅、T+1 无法止损），$P(\text{不成交})$ 又非零。相比之下，一个确定的小滑点**完全划得来**。

---

### Q3："成为流动性垫背但我仍然赚了钱"——只是赚少了吗？

你的静态视角确实没错，但在**动态风险**视角下有严重缺陷。

#### 静态视角（你的算法）

| 项目 | 数值 |
|------|------|
| 你的挂单价 | 10.00 |
| 成交价 | 9.90 |
| 瞬时"节省" | **+0.10** |

**结论**：只要明天开盘 > 9.90，你就是赢家。

#### 动态视角（机构的算法）

关键在**成交价为什么是 9.90**。

**统计事实**：在微观结构研究中，如果收盘集合竞价因为**大单砸盘**导致价格剧烈偏离，这种偏离**不是均值回归的噪音**，而是具有**持续的动量**。

原因：
1. **信息传导**：大单砸盘 = 卖方掌握某种信息 → 信息会在当晚/次日继续发酵
2. **技术面破位**：集合竞价低价成交 → 技术指标破位 → 次日引发止损盘
3. **T+1 的放大效应**：今天买的人明天才能卖，卖压积压
4. **情绪传染**：异常低的收盘价会在当晚讨论中被放大

**统计结果**：**次日跳空低开的概率显著上升**。

#### 一笔账

| 项目 | 数值 |
|------|------|
| 你的挂单价 | 10.00 |
| "优惠"成交价 | 9.90 |
| 节省 | **+0.10** |
| 次日开盘价（概率加权期望） | **9.70** |
| 次日实际损失 | **-0.20** |
| **净期望收益** | **-0.10** |

**关键分歧**：你把 0.10 看作**确定的利润**，机构把成交的**方式**看作**条件分布**的信号——成交价被砸越深，次日延续下跌的条件概率越高。

#### 本质上是"胜率"还是"期望收益"的问题？

都不是——是**两者同时恶化**。

| 对比 | 随机时点成交 | 流动性垫背成交 |
|------|-----------|--------------|
| 成交价 | 均值回归，无偏 | **左偏**（低价成交占比高） |
| 次日收益分布 | 对称 | **左偏 + 负漂移** |
| 胜率 | ≈ 50% | **< 50%** |
| 期望收益 | ≈ 0 | **< 0**（且尾部更重） |

**一句话**：你成交的那一刻，系统向你**发出了负面信号**——你接的是别人想扔的烫手山芋。

### 三个问题的统一答案

| 追问 | 散户直觉 | 机构的真实逻辑 |
|------|---------|--------------|
| 逆向选择是什么？ | 不知道这个概念 | 挂单 = 卖出免费期权给信息优势方 |
| 撤单不就平不了仓？ | 一根筋 | 撤单 ≠ 不成交，是为了**重构为主动单** |
| 垫背就是赚少了？ | 静态计算 | 成交方式是**条件分布信号**，次日期望为负 |

**串起来**：
> 收盘前撤单的本质不是"不想成交"，而是**"拒绝以含毒的方式成交"**。
> 机构用**确定的滑点**，换**免于逆向选择 + 免于隔夜黑天鹅 + 免于垫背信号**的三重保险。

## 八、反面视角：什么时候机构不撤单，甚至主动入场？

上面的分析如果读得过于绝对，会得出"机构全部跑光、14:57 后池子里只剩散户"的结论——这**不符合实际**。A 股每天收盘集合竞价的成交量通常**占全日 5%-15%**，其中相当一部分来自机构。

量化交易里没有"绝对"，只有"**概率**"和"**策略约束**"。机构的逃离是**条件性**的，以下三类资金恰恰**反向而行**，甚至**主动冲入**集合竞价。

---

### 8.1 场景 A：以收盘价为基准的策略（Benchmark Tracking）

**这是最大的一类不撤单力量。**

#### 指数基金 / ETF

- **KPI 是跟踪误差 (Tracking Error)**
- 指数是**怎么算收盘价的**，基金就必须**怎么买**
- 如果它们在 14:56 提前买入：
  - 最后三分钟股价**大涨** → 基金买在了便宜的价位，相对指数**正偏离** → 看似好事，实际会被监管/投资者质疑"主动择时"
  - 最后三分钟股价**大跌** → 基金买贵了，相对指数**负偏离** → 跟踪误差警报
- 二者都不被允许——基金的定位就是**无脑跟踪**，不能有任何择时成分

#### MOC (Market on Close) 算法

许多大资金（**主权基金 / 社保基金 / 保险资金**）在调仓时会明确要求执行 MOC 算法：

- 任务描述：**在收盘的那一刻，以那一刻的价格，完成成交**
- 不撤单是**纪律**
- 滑点与成本**不是评价标准**，**"成交价 = 当日收盘价"才是**

#### 特征指纹

这类资金的行为与 HFT 机构的"大逃亡"完全相反：
- 全程**稳定挂单**
- 14:56:59 **不撤单**
- 14:57 后挂单**仍在池中**等待撮合
- 订单量往往**巨大**（几千万到上亿）

---

### 8.2 场景 B：规模太大，集合竞价是"唯一的出口"

撤单后立刻市价单平仓的策略**只适用于中小型头寸**。超大头寸必须走集合竞价。

#### 流动性陷阱

- 机构持仓：某只票 **5,000 万元**
- 连续竞价最后一分钟的买盘：**总共才 200 万元**
- 如果在这一秒强平：
  - 直接把股价**打到跌停**
  - 造成巨大的**冲击成本**（参照 [平方根冲击定律](../Basics/SquareRootImpactLaw.md)）
  - 剩余 4,800 万根本卖不掉

#### 集合竞价的优势（对超大单而言）

- 集合竞价采用**最大成交量原则**
- 像一个**巨大的漏斗**，把买卖双方的意愿**在瞬间对撞**
- 对于超大单，集合竞价提供的**瞬时深度**远高于连续竞价
- 相当于：连续竞价是一个狭窄的水龙头，集合竞价是一个巨大的蓄水池

#### 机构的操作

- 不仅**不撤单**
- 还会**提前计算好价格**
- **死守**在池子里
- 可能配合大宗交易 / 盘后协议等工具联动

#### 特征指纹

- 单笔**超大订单**（千万级）提前挂入
- 挂单价格通常贴近**最大成交概率价位**（通过盘前算法预测）
- 一旦挂入，不会撤单
- 典型场景：**季末调仓、指数成分股调整、大股东减持**

---

### 8.3 场景 C：收割者策略（Liquidity Provision / Arbitrage）

**"如果大家都跑了，留下的就是机会。"**

这是反向利用 §4 "确定性溢价"现象的策略。

#### 反向算法的逻辑

- 专门**监控尾盘的"过度撤单"**
- 如果发现 14:56:59 撤单导致了价格被**异常压低**（超卖）
- 判断这是一个**非理性的流动性枯竭**——不是真实的负面信息，而是机构出于风控规则的被动撤离

#### 算法的决策

此时它会**留在集合竞价里作为买方**。它赌的是：

- 这种由"**机构逃跑**"造成的下跌是**暂时的**
- 明天早盘价格会**回归正常**
- 这时候它赚的就是散户以为赚到的那"**便宜的 1 毛钱**"
- 而且这 1 毛钱**足以覆盖风险**

#### 这与散户接盘的区别？

**信息/量化角度**：

| 对比 | 散户接"便宜单" | 收割者算法接"便宜单" |
|------|--------------|-------------------|
| 是否监控全市场撤单行为 | ❌ | ✅ 实时跟踪 `InstitutionalCancelSpike_t` |
| 是否能区分"信息驱动砸盘" vs "风控驱动撤单" | ❌ | ✅（通过成交量/新闻/相关资产联动） |
| 仓位大小 | 单票 | 组合对冲 |
| 次日止损机制 | 无 | 严格 |
| 期望收益 | **负**（信息劣势） | **正**（信息优势 + 对冲） |

同样是"接便宜货"，**是否具备分辨能力**决定了期望收益的正负号。散户和收割者算法都在接飞刀——但收割者会**读飞刀的轨迹**。

---

### 8.4 重新审视三个追问（反面视角）

如果机构选择**不撤单**，其实是做了一次权衡：

| 追问 | "撤单方"的理由 | "不撤方"的反驳 |
|------|-------------|-------------|
| **逆向选择风险** | 自己信息弱势，避开 | 自己才是知情者；或为了跟踪指数，**愿意支付这个风险成本** |
| **平仓风险** | 撤单后市价单 100% 成交 | 规模太大 → 集合竞价的**瞬时深度**反而更优 → 更低的总滑点 |
| **流动性垫背** | 接的是烫手山芋 | 我不是垫背，**我是收割恐慌红利的捕猎者** |

### 8.5 两条成交量曲线的叠加解释

为什么 A 股收盘集合竞价每天依然有**巨量成交**？

```
日内撤单率 vs 时间（概念图）

           │                                  ╱╲
           │                                 ╱  ╲
   撤单率   │                                ╱    ╲  ← HFT/日内机构"大逃亡"尖峰
           │          连续竞价背景水平        ╱      ╲
           │─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ╱       ╲___
           │________________________________________________
                                        14:56   15:00
                                        
日内挂单量 vs 时间（概念图）

           │                              ┌─────┐
           │                              │     │
  挂单量    │                              │     │   ← 指数/MOC/大资管"提前布阵"
           │                              │     │
           │________________________________________________
                                        14:56   15:00
```

**解释**：
- **尖峰撤单** = 高频 / 日内机构在"**夺门而出**"
- **池底平滑巨量** = 指数基金 / MOC / 超大持仓 / 收割者算法在"**冷眼旁观并接货**"
- 两者**并存不冲突**——它们属于完全不同的资金类型和策略约束

### 8.6 对订单归属识别研究的升级启示

最初的假设过于简化——**"集合竞价池内 = 散户"** 是错的。应升级为**二分类**：

| 集合竞价池内的参与者 | 指纹特征 |
|-------------------|---------|
| **散户**（最初的假设） | 零星挂单、订单金额小（< 5 万）、价格偏心理整数位、无前置行为 |
| **指数/MOC 机构** | **超大订单**（千万+）、价格接近 `vwap_today` 或 `indexed_tracking_price`、**盘前几小时就已下单** |
| **超大头寸出货** | 单笔**极大**（几千万+）、临近 14:56 才挂入、与大宗交易/席位信息有关联 |
| **收割者算法** | 14:57 之后在**被过度压低的价位**挂入买单、具备明显的**反向行为模式**、往往配对其他市场对冲（如期指） |

**研究启示**：识别散户不能只看"谁留在了池子里"，要结合：
1. **订单金额分布**（散户 vs 超大单二分法）
2. **挂单的提前量**（机构提前布阵 vs 散户临时起意）
3. **价格是否贴近 VWAP / 基准**（指数型机构的特征）
4. **是否配对其他资产**（收割者的对冲腿）

这将 `Fin-Research/OrderAttribution_AShare_2026Q2/` 主题的识别维度从**"六维"扩展到"七维"**——新增维度：**策略约束类型**（Strategy Constraint Type）。

> 图上的**平滑底座** = 大长线 / 指数资金在**冷眼旁观并精准下注**
