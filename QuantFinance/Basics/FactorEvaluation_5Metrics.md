# 因子评估的五大核心指标

在量化投资和因子挖掘中，仅仅构建出因子是不够的，必须通过一套严谨的统计和回测框架来证明其具有**稳定的盈利能力**且在**扣除交易成本后依然可行**。

> 前置：[IC、IR 与夏普比率](IC_IR_Sharpe.md)（概念层）| [IC/IR 计算详解](IC_IR_Calculation.md)（Pearson vs Spearman）| [t 统计量](T_Statistic.md)（显著性检验）| [滑点](Slippage.md)（交易成本）

---

## 1. Rank IC (秩信息系数)

**衡量目标**：因子的整体横截面预测能力。它回答了一个基本问题：**"因子值高（或低）的股票，未来的收益率就一定高（或低）吗？"**

### 公式

Rank IC 使用的是**斯皮尔曼秩相关系数 (Spearman Rank Correlation)**，而非皮尔逊相关系数。对于横截面上的 $N$ 只股票，计算它们在 $t$ 时刻的因子值排名，与 $t+1$ 时刻的未来收益率排名的相关性：

$$\text{Rank IC}_t = 1 - \frac{6 \sum_{i=1}^{N} d_i^2}{N(N^2 - 1)}$$

其中：
- $d_i$ 是第 $i$ 只股票的**因子值排名**与**预期收益率排名**的差值
- $N$ 是股票总数

### 为什么用 Rank IC 而不是普通 IC？

金融市场的数据充满了极端异常值（比如某只股票单日暴涨 20%）。普通 IC（皮尔逊相关系数）会被这些极端值**严重扭曲**，而 Rank IC 只看"名次"不看"绝对数值"，因此对极端值具有**极强的鲁棒性**。

> 详见 [IC/IR 计算详解](IC_IR_Calculation.md#4-为什么金融中几乎都用-rank-ic) 中的详细对比。

### 经验标准

| A 股日频因子 | 评价 |
|------------|------|
| 日均 $|\text{Rank IC}| > 0.05$ | **极强**的预测能力 |
| 日均 $|\text{Rank IC}| > 0.03$ | 有价值的**合格**因子 |
| 日均 $|\text{Rank IC}| < 0.02$ | 预测能力**可疑**，需进一步验证 |

## 2. ICIR (信息比率)

**衡量目标**：因子预测能力的**稳定性**。它回答了：**"这个因子是每天都能稳定赚一点点，还是靠几天的大赚掩盖了剩下时间的大亏？"**

### 公式

ICIR 是 Rank IC **时间序列**的均值除以其标准差：

$$\text{ICIR} = \frac{E[\text{Rank IC}_t]}{\sigma(\text{Rank IC}_t)}$$

年化处理（日频因子）：

$$\text{ICIR}_{\text{annualized}} = \text{ICIR} \times \sqrt{252}$$

### 现实意义

- **均值**代表预期收益
- **标准差**代表因子的波动风险
- 一个 Rank IC 均值很高但每天**剧烈震荡**（正负频繁横跳）的因子，在实盘中会面临巨大的回撤压力

> 与 [t 与 IR 的数学推导](T_IR_Derivation.md) 中的公式 $t \approx IR \times \sqrt{T}$ 一致——ICIR 本质上就是因子层面的信息比率。

### 经验标准

| 年化 ICIR | 评价 |
|----------|------|
| $> 2.0$ | **极其顶级**的因子 |
| $> 1.5$ | **优秀**的因子 |
| $1.0 - 1.5$ | 合格，可用于组合 |
| $< 1.0$ | 稳定性不足，需改进 |

> 高 ICIR 意味着你可以放心地对该策略**加杠杆**——因为收益/风险比足够高。

## 3. 多头年化超额 (Long-only Annualized Excess Return - Top 10%)

**衡量目标**：因子在**只做多 (Long-only)** 环境下的真实盈利上限。

### 公式

按因子值从大到小将股票分为 10 组（Deciles）。买入排名最靠前的 10% 股票（Top 10% 组），等权或市值加权构建组合。多头超额收益 = 组合收益 - 基准收益：

$$R_{\text{excess}, t} = R_{\text{Top10\%}, t} - R_{\text{benchmark}, t}$$

年化处理：

$$\text{Annualized } R_{\text{excess}} = (1 + R_{\text{cumulative}})^{252/T} - 1$$

### 为什么这个指标至关重要？

Rank IC 评估的是全市场的线性关系，但**实盘中我们不可能买下全市场 5000 只股票**——我们只会买得分最高的那一小撮。

有些因子的 Rank IC 很好，是因为它对**垃圾股（Bottom 10%）的暴跌**预测得很准，但对**好股票（Top 10%）的预测**却毫无卵用。

在 A 股，融券做空**成本极高且券源不足**，因此**"多头端是否有显著超额"是决定因子能否落地的生死线**。

> 这正是 [中美量化策略异同](../Markets/QuantStrategy_AvsUS.md) 中提到的 A 股特殊性——由于 T+1 和做空限制，A 股量化几乎只能做多头策略或用股指期货做不完美的对冲。

## 4. 多空夏普比率 (Long-Short Sharpe Ratio)

**衡量目标**：剥离市场大盘涨跌 (Beta) 后，因子**纯粹的风险调整后收益 (Alpha)**。

### 公式

构建一个**资金中性**的多空组合：做多 Top 10% 组，同时做空 Bottom 10% 组。计算该对冲组合的日收益率，然后求夏普比率：

$$\text{Sharpe}_{\text{LS}} = \frac{E[R_{\text{Top10\%}} - R_{\text{Bottom10\%}}] - R_f}{\sigma_{R_{\text{Top10\%}} - R_{\text{Bottom10\%}}}} \times \sqrt{252}$$

（在日频量化评估中，无风险利率 $R_f$ 往往简化为 0）

### 现实意义

这是评估一个因子**"纯净度"**的终极指标。

如果大盘处于疯牛市，哪怕扔飞镖选股也能赚钱，多头收益可能很高——但多空夏普会**剔除这种市场红利**。

> 多空收益曲线如果能够像**一条直线一样稳定以 45 度角向上**，是量化研究员的终极追求。

### 经验标准

| 扣费前多空年化夏普 | 评价 |
|----------------|------|
| $> 3.0$ | 因子具有**极强的独立 Alpha** |
| $2.0 - 3.0$ | 优秀因子 |
| $1.0 - 2.0$ | 合格，可用于因子池 |
| $< 1.0$ | Alpha 不足，扣费后可能为负 |

## 5. 因子自相关系数 / 双边换手率

**衡量目标**：因子的衰减速度与实盘交易成本（滑点与手续费）的代理指标。它回答了：**"为了赚这个因子的钱，我每天需要交多少'过路费'？"**

### 5.1 自相关系数 (AR-1)

衡量今天的因子暴露与昨天的因子暴露的相关性：

$$\rho_{\text{auto}} = \text{Corr}(\text{Factor}_t, \text{Factor}_{t-1})$$

- $\rho_{\text{auto}}$ 越高 → 因子变化越慢 → **换手率越低**
- $\rho_{\text{auto}}$ 越低 → 因子每天剧烈翻转 → **换手率爆表**

### 5.2 双边换手率

假设按该因子构建组合，每天权重的绝对变化量之和：

$$\text{Turnover}_t = \sum_{i=1}^{N} |w_{i,t} - w_{i,t-1}|$$

### 为什么这个指标是"拦路虎"？

这是从**"理论纸面富贵"走向"实盘真金白银"**的拦路虎。

很多基于高频盘口数据的因子（例如单纯的买卖力量失衡），其预测能力极强，但**信号变化极快**。今天因子值排第一的股票，明天可能就掉到了倒数第一。这就导致每天的换手率可能高达 **100% 甚至 200%**。

一旦把 A 股的双边交易成本（印花税 + 佣金 + 滑点，通常按**万分之 15 到千分之 1** 估算）扣除，高换手因子的超额收益**瞬间就会变为负数**。

> 这与 [A 股高频因子工程](../Research/AShare_HFFactorEngineering.md) 中讨论的 **Alpha 衰减 (Alpha Decay)** 是同一个问题——高频因子的半衰期可能只有秒到分钟级，在 T+1 下落地需要极其精细的换手率管理。

### 优化思路

高频因子必须做**时间平滑 (Time-series Smoothing)** 或引入**延迟惩罚**：
- 通过提高自相关系数来**强制降低换手率**
- 代价是牺牲一部分 Rank IC
- 但扣除交易成本后的净收益反而更高

> 这本质上是一个**Alpha 与交易成本的优化权衡**——目标不是最大化 Rank IC，而是最大化 $\text{Rank IC} - c \cdot \text{Turnover}$。

## 五大指标的关系与全景视图

```
Rank IC (准不准)
    ↓
ICIR (稳不稳)
    ↓
├── 多头超额 (多头端赚不赚)   ← A股的生死线
│
├── 多空夏普 (纯Alpha够不够)  ← 理论上限
│
└── 换手率 (过路费多不多)     ← 实盘落地的拦路虎
```

| 指标 | 回答的问题 | 关注阶段 |
|------|-----------|---------|
| **Rank IC** | 因子准不准？ | 因子挖掘 |
| **ICIR** | 因子稳不稳？ | 因子筛选 |
| **多头超额** | 只做多能赚吗？ | **A 股实战生死线** |
| **多空夏普** | 纯 Alpha 够吗？ | 理论评价 |
| **换手率** | 扣费后还赚吗？ | **实盘落地** |

> **实践中的筛选流程**：先用 Rank IC > 0.03 和 ICIR > 1.5 过滤出候选因子 → 再看多头 Top 10% 超额是否显著 → 最后检查换手率在合理水平（如日均 < 30%）时扣费后是否仍有利润。只有同时通过所有关卡的因子才会进入最终的组合。
