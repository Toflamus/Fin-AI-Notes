# t 统计量 (t-Statistic)

t 统计量用来衡量一个预测变量对结果的影响是否"真实"——是确有其事，还是仅仅因为运气好凑出来的。

---

## 1. 核心概念

在回归模型中，每个自变量会得到一个系数 $\hat{\beta}$。t 统计量验证**零假设**：该系数是否等于 0。

$$t = \frac{\hat{\beta} - 0}{SE(\hat{\beta})} = \frac{\text{信号}}{\text{噪声}}$$

- **分子**：你的发现（估计值，即信号强度）
- **分母**：抽样带来的随机波动（标准误差，即噪声水平）
- **判断标准**：$|t| > 1.96$（95% 置信水平）→ 变量**显著**，由随机偶然产生的概率 < 5%

---

## 2. 在金融研究中的应用

金融数据（尤其高频）噪声极强，t 统计量是核心的"测谎仪"。

### 2.1 验证因子有效性

发现 OFI 与下一秒价格变动正相关，但这是稳定规律还是偶然？

- t 值高 → 该因子对价格的影响在统计上稳健，是真信号
- t 值低 → 可能只是噪声，不值得纳入模型

### 2.2 与夏普比率 / 信息比率的关系

$$t \approx \text{IR (信息比率)} \times \sqrt{T \text{ (样本量)}}$$

含义：
- 即使策略收益率低，只要极其**稳定**且样本量够大，t 值也会高
- 研究者通过 t 值判断一个因子是否值得投入真金白银
- 这也是为什么高频策略（样本量巨大）更容易产出高 t 值因子

### 2.3 变量筛选与防过拟合

构建模型时可能纳入 OFI、成交量、买卖价差等多个变量。通过各变量的 t 值：
- **高 t 值** → 保留，统计显著
- **低 t 值** → 剔除，"看起来有用但统计上不显著"，防止过拟合

### 2.4 大样本下的稳健性

金融数据往往不服从正态分布（肥尾），但在大样本下（高频数据每秒成千上万个数据点），根据**中心极限定理**，t 统计量依然是可靠的判别工具。

---

## 3. 速记

| 概念 | 说明 |
|------|------|
| t 值本质 | 信号 / 噪声 |
| 显著性阈值 | \|t\| > 1.96（95% 置信） |
| 零假设 | $\beta = 0$（该变量无影响） |
| 金融用途 | 因子验证、策略评估、变量筛选 |
| 与 IR 的关系 | $t \approx \text{IR} \times \sqrt{T}$ |
