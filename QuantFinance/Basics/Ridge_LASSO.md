# Ridge 回归与 LASSO 回归

Ridge（岭回归）和 LASSO（套索回归）本质上都是对 [OLS](OLS_MultipleRegression.md) 的改进。当数据集中存在**多重共线性**（特征之间高度相关），或者**特征数量多于样本数量**（$k > n$）时，OLS 的矩阵 $X^TX$ 会接近奇异矩阵甚至不可逆，导致 OLS 估计的系数变得极其不稳定，方差极大（即模型容易过拟合）。

核心思想：在 OLS 的损失函数（残差平方和 SSR）后面加上一个**惩罚项（Regularization Term / 正则化项）**，用来限制回归系数的大小。

> 相关：[多元 OLS](OLS_MultipleRegression.md) | [特征净化与降维](../DataProcessing/FeatureOptimization.md)（ZCA/PCA 解决共线性的另一条路）| [协方差/PCA/ZCA 数学原理](../DataProcessing/Covariance_PCA_ZCA_Math.md)

## 1. Ridge Regression (岭回归 — L2 正则化)

岭回归通过对系数的**平方和**进行惩罚，来控制模型的复杂度。

### 1.1 优化目标函数

$$\text{Loss}_{\text{Ridge}} = \sum_{i=1}^{n}(y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^{k} \beta_j^2$$

矩阵形式：

$$\text{Loss}_{\text{Ridge}} = (Y - X\beta)^T(Y - X\beta) + \lambda \beta^T \beta$$

| 组成部分 | 含义 |
|---------|------|
| $(Y - X\beta)^T(Y - X\beta)$ | 与 OLS 完全相同的残差平方和（拟合好坏） |
| $\lambda \beta^T \beta = \lambda \sum \beta_j^2$ | **L2 惩罚项**：系数平方和（模型复杂度） |
| $\lambda$（Lambda） | **正则化强度参数**（超参数，$\lambda \geq 0$） |

### 1.2 $\lambda$ 的作用

| $\lambda$ 取值 | 效果 |
|---------------|------|
| $\lambda = 0$ | 惩罚项消失，退化为普通的 OLS |
| $\lambda$ 适中 | 在拟合能力与模型复杂度之间取得平衡 |
| $\lambda \to \infty$ | 惩罚项无穷大，迫使所有系数 $\beta_j \to 0$ |

### 1.3 解析解

对 $\text{Loss}_{\text{Ridge}}$ 关于 $\beta$ 求导令其为零（推导过程与 OLS 完全类似，只是多了 $\lambda \beta^T \beta$ 项的导数 $2\lambda \beta$）：

$$\frac{\partial \text{Loss}}{\partial \beta} = -2X^TY + 2X^TX\beta + 2\lambda \beta = 0$$

$$-2X^TY + 2(X^TX + \lambda I)\beta = 0$$

$$(X^TX + \lambda I)\beta = X^TY$$

$$\boxed{\hat{\beta}_{\text{Ridge}} = (X^TX + \lambda I)^{-1} X^TY}$$

**与 OLS 对比**：

| | OLS | Ridge |
|---|-----|-------|
| 闭式解 | $(X^TX)^{-1}X^TY$ | $(X^TX + \lambda I)^{-1}X^TY$ |
| 可逆性 | 可能不可逆（共线性） | **必然可逆**（$\lambda I$ 保证正定） |

> **核心直觉**：$\lambda I$ 相当于在 $X^TX$ 的对角线上加上了一个正数。这在数学上**保证了矩阵必然可逆**，完美解决了多重共线性的问题。这也是"岭回归"名字的来源——在矩阵的"山脊（ridge/对角线）"上加了一个常数。

### 1.4 特点

- Ridge 会把所有特征的系数"往零的方向压缩"（**Shrinkage**）
- 但**永远不会把它们完全压缩为 0**
- 因此 Ridge **不能做特征选择**，所有特征都会被保留在模型中
- 适合所有特征都可能有用、但需要抑制过拟合的场景

## 2. LASSO Regression (套索回归 — L1 正则化)

LASSO（**L**east **A**bsolute **S**hrinkage and **S**election **O**perator）通过对系数的**绝对值之和**进行惩罚。

### 2.1 优化目标函数

$$\text{Loss}_{\text{LASSO}} = \sum_{i=1}^{n}(y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^{k} |\beta_j|$$

| 组成部分 | 含义 |
|---------|------|
| $\sum(y_i - \hat{y}_i)^2$ | 残差平方和（与 OLS 相同） |
| $\lambda \sum |\beta_j|$ | **L1 惩罚项**：系数绝对值之和 |

### 2.2 无闭式解

由于绝对值函数 $|\beta_j|$ 在零点处**不可导**，LASSO 没有像 OLS 和 Ridge 那样的简单闭式解析解。

通常需要用迭代算法求解：
- **坐标轴下降法（Coordinate Descent）**：逐个坐标优化，是最常用的方法
- **LARS（Least Angle Regression）**：高效的路径算法
- **次梯度法（Subgradient Method）**

### 2.3 核心特点：稀疏性与特征选择

由于 L1 惩罚项的几何性质（下文详解），随着 $\lambda$ 的增大，LASSO 会直接把那些不重要的特征的系数**精确地变成 0**。

因此，LASSO 自带**特征选择 (Feature Selection)** 的功能：
- 能够输出一个**稀疏模型**（只有少数非零系数）
- 非常适合处理**高维数据**（特征数远大于样本数）
- 自动筛选出最重要的特征

---

## 3. 几何直觉：为什么 LASSO 能把系数变成 0 而 Ridge 不能？

这是理解两者差异的**最核心**的直觉。

### 3.1 等价的约束优化问题

我们可以把带惩罚项的优化问题转化为**带约束条件的极值问题**：

- **OLS 损失函数**：在二维参数空间 $(\beta_1, \beta_2)$ 中，表现为以 OLS 最优解 $\hat{\beta}_{\text{OLS}}$ 为中心的一组**椭圆形等高线**。越靠近中心，损失越小。

- **Ridge 的约束条件**：$\beta_1^2 + \beta_2^2 \leq t$，在平面上是一个**圆形区域**

- **LASSO 的约束条件**：$|\beta_1| + |\beta_2| \leq t$，在平面上是一个**菱形区域**（正方形旋转 45 度）

### 3.2 优化过程的几何图景

优化的过程就是让椭圆不断膨胀，直到**第一次触碰到约束区域的边界**。这个触点就是最终的解。

**对于 Ridge（圆形约束）**：

- 圆形的边界处处光滑，**没有尖角**
- 椭圆与圆形的切点几乎不可能刚好落在坐标轴上
- 所以得到的 $\beta_1$ 和 $\beta_2$ 都会很小，**但都不是 0**
- → Ridge 只压缩，不归零

**对于 LASSO（菱形约束）**：

- 菱形有**尖锐的顶点**（这些顶点恰好在坐标轴上）
- 膨胀的椭圆**非常容易**第一次就碰到这些顶点
- 一旦触碰在顶点上，意味着某一个维度的系数刚好为 0（比如 $\beta_1 = 0, \beta_2 \neq 0$）
- → LASSO 实现了稀疏性

```
        Ridge (圆形约束)              LASSO (菱形约束)

         β₂                            β₂
          |   ╱╲                         |   /\
          |  ╱  ╲                        |  /  \
          | ╱ ●  ╲  ← OLS 解            | / ●  \  ← OLS 解
          |╱ ╭──╮ ╲                     |/      \
     ─────●──│  │──●── β₁         ─────●────────●── β₁
          |╲ ╰──╯ ╱                     |\      /
          | ╲    ╱                       | \    /
          |  ╲  ╱                        |  \  /
          |   ╲╱                         |   \/
                                              ↑
    切点不在轴上                        切点在顶点上
    → β₁ ≠ 0, β₂ ≠ 0                → β₁ = 0! (稀疏)
```

> **数学本质**：L1 范数的等高面（菱形）在坐标轴方向有"尖点"（不可微点）。正是这些尖点与 SSR 椭圆的"频繁相遇"，造就了 LASSO 的稀疏性。L2 范数的等高面（圆形）处处光滑，没有尖点，所以解总是在"一般位置"上，不会精确落在轴上。

## 4. 对比速查表

| 维度 | OLS | Ridge (L2) | LASSO (L1) |
|------|-----|------------|------------|
| 惩罚项 | 无 | $\lambda \sum \beta_j^2$ | $\lambda \sum |\beta_j|$ |
| 约束几何 | 无约束 | **圆形** | **菱形** |
| 闭式解 | $(X^TX)^{-1}X^TY$ | $(X^TX + \lambda I)^{-1}X^TY$ | **无**（需迭代算法） |
| 多重共线性 | 崩溃 | **解决**（$\lambda I$ 保证可逆） | 部分解决（通过选择一个特征丢弃其余） |
| 系数压缩 | 无 | 压缩但**不归零** | 压缩且**可以精确归零** |
| 特征选择 | 不做 | **不做** | **自动做** |
| 适用场景 | 特征少、无共线性 | 特征都可能有用、需抑制过拟合 | 高维稀疏、需要自动筛选特征 |
| 在量化中的位置 | 基础基准 | 中性化回归、风险模型 | 因子筛选（从数千候选因子中选出有效的） |

## 5. Elastic Net：两者的折中

如果既想要 LASSO 的特征选择能力，又想要 Ridge 在共线性特征组中的稳定性，可以使用**弹性网 (Elastic Net)**：

$$\text{Loss}_{\text{Elastic Net}} = \sum(y_i - \hat{y}_i)^2 + \lambda_1 \sum|\beta_j| + \lambda_2 \sum \beta_j^2$$

Elastic Net 同时使用 L1 和 L2 惩罚。通过调节 $\lambda_1$ 和 $\lambda_2$ 的比例，可以在纯 Ridge 和纯 LASSO 之间滑动。

> **实践建议**：在量化因子筛选中，通常先用 LASSO 做**初步特征选择**（从数千个候选因子中筛选出一小批），然后用 Ridge 或 Elastic Net 对筛选后的因子做**最终的回归建模**。这种两阶段流程兼顾了特征选择和系数稳定性。
