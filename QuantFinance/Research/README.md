# Research — 文献调研与研究笔记

文献驱动的研究笔记。每读一篇论文或做一次调研，在这里留下一份结构化的笔记。

## 笔记结构规范

每份笔记建议包含以下部分：

1. **一句话总结** — 这篇在说什么
2. **核心贡献 / 核心发现** — 3-5 个要点
3. **关键公式与推导** — 完整保留，不省略中间步骤
4. **A 股适用性 / 实战落地** — 在 A 股环境下是否适用、如何修正
5. **与知识库其他内容的关联** — 交叉引用
6. **待深入的问题** — 读完后产生的新问题，指导下一步阅读
