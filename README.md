# Knowledge Base - 量化金融 & 人工智能

本文档库用于系统性地整理和归纳**量化金融**与**人工智能**两大领域的学习笔记与知识总结。

## 内容来源

- 与 AI 的对话记录中提炼的知识点
- 书籍、论文中的重要内容摘录
- 实践中积累的经验与心得
